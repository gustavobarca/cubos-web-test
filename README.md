Projeto para o desafio técnico de web da Cubos Academy.

### 🧩 Tecnologias usadas

- ⚛️ React JS
- 💅 Styled Components
- 🎣 React Hooks

###  🚀 Iniciando

Clone o projeto:

    git clone https://github.com/gustavobarca/cubos-web-test
    
Acesse a pasta do projeto:

    cd cubos-web-test

Instale as dependências:

    yarn

Inicie o projeto

    yarn start
